# `figures/` — Figure regeneration scripts

Every paper figure can be regenerated by running its script from the repo root.
All scripts read inputs from `data/` via repo-relative paths and write outputs
to `figures/out/`.

```bash
python figures/ad_figure_trap1_CCN.py
python figures/ad_figure_trap2_CCN.py
# ...
```

Or regenerate everything at once:

```bash
for f in figures/*.py; do python "$f"; done
```

## Data dependencies

All inputs come from `data/`. The scripts use `REPO = Path(__file__).resolve().parent.parent`
to resolve paths, so they run from any cwd.
